## ✨ Features

- 🛡️ **Security header audit** — CSP, HSTS, X-Frame-Options, X-Content-Type-Options and more
- 🌐 **WHOIS & registration data** — domain age, registrar, and ownership signals
- 🔒 **SSL certificate check** — validity, issuer, expiry, and chain integrity
- ⚡ **DNS record lookup** — A, MX, TXT, SPF, DMARC records in one view
- 📊 **Trust score** — composite rating that surfaces red flags immediately
- 🔁 **Redirect chain tracer** — see every hop a URL takes before landing
- ⚠️ **Plain-English risk summary** — no technical jargon, just clear signals

## 🚀 Quick Start

```bash
git clone https://github.com/jayblast-spec/trustsignal.git
cd trustsignal
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
